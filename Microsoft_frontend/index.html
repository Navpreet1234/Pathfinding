A* algorithm

A* Algorithm : A* is an informed search algorithm, meaning that it is formulated in terms of weighted graphs i.e. starting from a specific starting node of a graph, it aims to find a path to the given goal node having the smallest cost.

Breadth First Search Algorithm : In this algorithm, we start at the tree root and explore all of the neighbor nodes at the present depth prior to moving on to the nodes at the next depth level.

Dijkstra algorithm

Dijkstra's Algorithm : In this algorithm, for a given source node we find the shortest path between that node and every other. Hence, it can be used for finding the shortest paths from a single node to a single destination node by stopping the algorithm once the shortest path to the destination node has been determined.
